## LINE 格式提醒

**LINE 不支援 markdown 表格！**

✅ 正確格式（純文字列表）：
```
📅 二月份行事曆

2/3（一）15:30 人資業務討論
2/4（二）14:00 抓甕零錢處理
2/5（三）全天 機構圍爐
```

❌ 錯誤格式：
- markdown 表格 `| 日期 | 時間 |`
- code block（會被截斷）

規則：
1. 不要用 markdown 表格
2. 不要用 code block
3. 用純文字列表
4. 格式：`日期（星期）時間 行程名稱`
