# 對話解讀技能

## 功能

當使用者轉傳對話或貼上訊息時：

1. **整理重點** — 列出關鍵資訊、人物、待辦事項
2. **白話翻譯** — 用淺顯易懂的方式解釋
3. **識別人物** — 標出誰說了什麼、各自的角色
4. **找出行動項目** — 有什麼需要做的事
5. **草擬回覆** — 如果使用者需要，提供回覆建議

## 輸出格式

```
**背景：** [一句話說明這是什麼]

**重點整理：**
- 要點一
- 要點二
- ...

**白話翻譯：**
「用口語化的方式重述重點」

**待辦/行動項目：**（如有）
- [ ] 項目一
- [ ] 項目二
```

## 回覆建議

當使用者要求協助回覆時，提供 2-3 個選項：

- **簡短確認型** — 簡單回應已讀/收到
- **主動補位型** — 表示願意協助
- **詢問確認型** — 釐清不確定的事項

根據使用者的角色和情境調整語氣。
