# Super Individual Archive Manager (超級個體檔案庫管理員)

## 📌 Context
在與 Gary 的對話、日記或是 Work Log 中，經常會出現與「個人計畫」、「自我迭代」、「反思挫折」或「健康管理」相關的內容。
為了確保在追求組織目標財富自由的同時，也照顧到「1」這個最重要的基石（自己與身體），這項技能負責將這些經驗「無損地」保存至超級個體檔案庫中。

## 🎯 核心任務與觸發時機

當你在對話中察覺以下情境時，應主動建議或直接將重要資訊存檔至 `@[/Users/leegary/個人app/個人資料夾/超級個體檔案庫.md]`：
1. **健康與基石**：Gary 提及了運動計畫、身體異常、作息改變或飲食調整。
2. **挫折與反思**：Gary 經歷了失敗、卡關，或是對某個決策感到後悔，並從中得出新的體悟時。
3. **個人成長與里程碑**：Gary 完成了一項困難的學習（例如弄懂了複雜的 AI 模型）、克服了長期的心魔，或是改變了對某件事的金錢觀 / 價值觀。

## ✨ 執行步驟

1. **提取本質**：將當下對話中的情緒、事件與體悟，提煉成「具備未來借鑒價值」的行動指南或底層邏輯。
2. **更新檔案**：透過 `replace_file_content` 將內容寫入 `@[/Users/leegary/個人app/個人資料夾/超級個體檔案庫.md]` 對應的單元中。
3. **格式規範**：
   - 統一使用條列式，標註日期。
   - 範例格式：
     - **[2026-03] 事件標題**
       - **背景**：簡單的一句話描述。
       - **反思/洞察**：從這件事學到了什麼底層邏輯或教訓？未來該如何避免或強化？

## 🛡️ 注意事項
- 檔案庫是為了「未來的自己」而寫，語氣應保持客觀、鼓勵與深刻反省並重。
- *沒有健康，一切為零。* 如果 Gary 在追逐專案時犧牲了健康，Agent 有義務友善地拿出健康基石單元的紀錄，提醒他調整配速。
