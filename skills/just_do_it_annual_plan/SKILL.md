# 著陸式年度計畫法（Landing-Style Annual Plan）

> 出自邹小强《只管去做》——整個過程是「從上到下」制定年度計畫，「從下到上」實現年度計畫。

## 核心理念

年度計畫落實的過程，就像飛機從萬米高空著陸：

| 高度 | 對應層級 | 管理類型 |
|------|---------|---------|
| 萬米高空（年） | 願景 → 目標 | **目標管理** |
| 五千米（月） | 項目計畫 + 習慣計畫 | **計畫管理** |
| 跑道（日） | 4D 工作法 | **精力管理** |

---

## 第一步：寫一封給五年後自己的信（願景）

### 目的
找到讓自己「怦然心動」的願景，這是年度計畫的動力源頭。

### 三個要求
1. **用五感寫作法**，不要列「乾條條」
   - ❌ 每週去健身房兩次
   - ✅ 絕對沒有小肚腩，跟20多歲年輕人在球場上身體對抗完全不落下風
2. **不能只寫感受**（空想家模式）
   - ❌ 你一定過得很幸福吧……
   - ✅ 要有具體的畫面感、有意義的細節
3. **當作一個儀式**
   - 找不被打擾的時間段、舒服放鬆的環境、喜歡的筆和紙

### 五感寫作法（去腦存心）
放慢速度，先運用五感（視覺、聽覺、味覺、觸覺、嗅覺）→ 用心感受 → 再走腦。
速度太快 = 從五感直接走腦 = 走心失敗。

## 第二步：目標九宮格

### 八大領域

```
┌──────────┬──────────┬──────────┐
│ 學習成長  │ 體驗突破  │ 休閒娛樂  │  ← 三種享受（頭頂）
├──────────┼──────────┼──────────┤
│ 工作事業  │ 年度主題詞│ 家庭生活  │  ← 工作與生活平衡（左右手）
├──────────┼──────────┼──────────┤
│ 身體健康  │ 財務理財  │ 人際社群  │  ← 三個根基（腳下）
└──────────┴──────────┴──────────┘
```

### 九宮格像一個人
- **腳下三根基**：健康、財務、社交（沒有這些什麼都做不了）
- **左右手**：工作與生活的平衡
- **頭頂三享受**：成長享受、愉悅享受、放鬆享受
- **中間**：年度主題詞（如「蛻變」「創造現金流」）

### 核心理念
> 「參差多態乃幸福本源」——羅素。九宮格的目的是讓年度目標更**平衡**。

### 體驗突破
做一些以前從來沒有做過的事情，看看會發生什麼。
> 「人生除了活得有意義，還要活得有意思！」

## 第三步：把願景孵化為目標

### 三步孵化法
1. **先定義**：這個願景到底是什麼意思？（例：財務自由 = 被動收入 > 總支出）
2. **再計畫**：為了實現這個定義，我打算怎麼做？
3. **後目標**：明年的具體目標是什麼？

### 目標可行性檢驗

#### 填空題：時間預算
- 實現年度目標所需時間 = _____ 小時
- 一年中可支配時間 = _____ 小時
- **每天可支配時間 ≈ 2小時**（扣除通勤、吃飯、娛樂等）
- 一年 ≈ 52週 × 14小時/週 ≈ **728小時**

#### 判斷題：SMART 原則
- **S**pecific（具體）
- **M**easurable（可衡量）
- **A**ttainable（可達成）
- **R**elevant（與其他目標相關）
- **T**ime-based（有時間期限）

> ⚠️ SMART 只適合衡量**事情**，不適合衡量**人際關係**

#### 做一個現實的理想主義者
> 「所有年底眼睛裡流的淚，都是年初腦子裡進的水。」

## 第四步：項目計畫

### 項目 vs 習慣
- **項目**：多個步驟完成的事情（如出書、旅行）
- **習慣**：週期性去做的事情（如每天閱讀、運動）

### 抬頭看路：甘特圖（月度）
1. **畫圖**：A4橫放，12個月刻度 + 左側列出所有項目
2. **填充**：用空心方框標示每個項目的預計起止時間
3. **調整**：每月最後一天更新
   - 正常推進 → 塗成實心
   - 超前完成 → 多塗一段
   - 進度滯後 → 塗少一點（提醒加速）

### 低頭走路：倒推分解法
像馬拉松一樣，從終點倒推每個標誌性節點，**一直倒推到「現在」**。

越接近現在，行動計畫越要明確具體。

### 項目四象限
| | 具體 | 模糊 |
|---|---|---|
| **簡單** | 「預備，跑」| 保持探索、彈性 |
| **複雜** | 標準倒推分解 | **需要項目化解** |

### 項目化解 vs 項目分解
- **項目分解**：明確完成步驟（What）
- **項目化解**：解決步驟中的難題（How）

#### 化解兩方向
1. **讓它變簡單**（複雜 → 簡單）
2. **讓它變具體**（模糊 → 具體）

#### 化解技巧
- 列三級大綱（模糊→具體）
- 散步收集靈感（放鬆產生好想法）
- 找兩個朋友碰撞（一個懂行的 + 一個受眾）

### 好的項目計畫三標準
1. **坚韌**：接不完美的自己，留緩衝時間、做B計畫
2. **簡化**：「簡」讓複雜回歸簡單，「化」化解難題
3. **解放**：讓事務擺脫對大腦的糾纏

### 舒適區-學習區-恐慌區配比
建議目標配比 **3-6-3**：3個舒適區 + 6個學習區 + 3個恐慌區

## 第五步：習慣培養

### 培養習慣四階段
1. **新鮮感和熱情開始**（三分鐘熱度在此階段是正常的）
2. **用方法和技巧堅持**
3. **吸納渴求和獎賞的能量來突破**
4. **伴隨喜悅的習慣成自然**

### 習慣回路（出自《習慣的力量》）
```
暗示 → 惯常行為 → 獎賞
       ↑               |
       └── 渴求驅動 ──┘
```
> 習慣不能被消除，只能被**替代**（改變惯常行為，保留暗示和獎賞）

### 找到真正想養成的習慣：評分法

| 習慣 | 痛苦(0-10) | 熱情(0-10) | 時機(0-10) | 總分 |
|------|-----------|-----------|-----------|------|
| 習慣A | ? | ? | ? | ? |
| 習慣B | ? | ? | ? | ? |

- **痛苦**：解決我哪個痛苦？那個痛苦有多深？
- **熱情**：做這件事的動力有多強烈？
- **時機**：眼下適不適合做這件事？

> 時間管理真的不是在做加法，而是在做**減法**！

### 習慣培養卡片（啟動儀式）

```
為了 [內心渴求/初心]，
我 [觸發時間] 後，
原來是 [舊行為]，
現在是 [新習慣行為]。

成功標準：連續 ___ 天完成
成功獎勵：___

如果 [預想阻礙1]，那麼 [應對方式1]
如果 [預想阻礙2]，那麼 [應對方式2]
如果我想放棄，那麼 [求助方式]
```

### 循序漸進的計畫（不要一蹴而就）
- 困難不在量（如走一萬步很簡單），而在「每天」
- 先連續一週 → 再連續三週 → 最終養成
- 像遊戲打怪一樣，一級一級突破

### 三條命機制
- 每月允許 3 次「掛掉並復活」
- 用完三條命才 Game Over，重新開始
- **至剛則易摧**，坚韌的計畫比「一定要」的計畫更靠譜

### 忠實記錄 + 積極反饋
- 用表格記錄每日完成情況
- 關注積極面（成功18天 vs 失敗1天 → 非常成功！）
- **內心驅動力 > 外在誘惑力**
  - 獎賞 = 過程中的內在感受（如平靜、靈感、被認可）
  - 獎勵 = 達成後的外在物品（僅在習慣培養成功後給）

### 習慣的自生自滅
- 養成後放手，讓它自己運行
- 把注意力轉移到培養下一個習慣

## 第六步：4D 每日工作法

### 4D 決策框架
| D | 含義 | 行動 |
|---|------|------|
| **Do it now** | 立即去做 | 寫在自由區，優先完成 |
| **Delay it** | 計畫去做 | 分四類處理（見下） |
| **Delegate it** | 授權去做 | 確認回覆時間，設提醒跟進 |
| **Don't do it** | 盡量別做 | 不寫入清單 |

### Delay 四類
1. 今天做 + 固定時間 → 手機日曆
2. 今天某時間段做 → 左側時間軸（硬性時間）
3. 今天做但隨時可做 → 右邊自由區 + 任務分解
4. 將來做 → 寫到月曆

### 每日待辦清單格式
```
┌─────────────┬───────────────────────┐
│  時間軸      │     自由區             │
│  (硬性時間)   │  (待辦事項+任務分解)    │
│             │                       │
│ 9:00 開會    │ ★重要任務              │
│ 10:00       │   - 步驟1              │
│ 14:00 面試   │   - 步驟2              │
│             │   - 步驟3              │
│             │ 背單詞60個（年度計畫）   │
│             │ 其他任務...             │
└─────────────┴───────────────────────┘
```

### 關鍵原則

1. **工作、生活、年度計畫寫在同一張清單上**——它們是平等的
2. **任務分解**：利用碎片時間（10-15分鐘）完成大任務的子步驟
3. **寫下做事思路**：減少分心（被打擾一次 → 平均13分鐘才能重新專注）
4. **間作套種法**：腦力與體力交替、抽象與具象交替

### 付氏工作流（應對臨時突發事件）
```
事情出現 → 2分鐘能解決？→ 是 → 立刻做
                         → 否 → 寫下來
                                  ↓
                              預覽所有事情
                                  ↓
                              安排輕重緩急
                                  ↓
                         優先做重要的事情
                                  ↓
                     非即時交流有空再處理
```

### 五個臨時事件處理原則
1. **排序依據價值觀**：沒有對錯，但要承擔後果
2. **2分鐘原則**：2分鐘能解決就立刻做
3. **刺激與反應之間有空隙**：立即回應 ≠ 立即去做
4. **先預覽再排序**：用假設排序不如先確認事實
5. **寫下來**：所有事情在紙上，比在腦中更有條理

## 時間管理三類

| 類型 | 時態 | 方法舉例 |
|------|------|---------|
| 計畫 | 未來 | 衣櫃整理法 |
| 執行 | 現在 | 番茄工作法 |
| 反思 | 過去 | 時間日誌 |

> 過去、現在、未來，都是在為**目標**做準備。

## 結語金句

> 「做事靠系統」——系統讓你實現年度計畫的概率大增。

> 三分鐘熱度是正常的，靠熱度把習慣培養起來才是不正常的。

> 培養習慣的燃料：**熱情**（內心需要 + 賦予更深的意義）和 **痛苦**。

> 內心的驅動力比外在的誘惑力輸出更持久的能量。

> 時間管理也好、習慣培養也好，其實是一門**心理學**。
