# Keyword Manager (關鍵字/標籤管理員)

## 核心功能
本 Skill 負責維護全系統通用的關鍵字列表，供「參考資料歸類」與「任務分類」時優先對照使用。

## 現有關鍵字列表 (Keyword Registry)
*這些關鍵字將優先用於 Things 3 Tags 或 Notion 標籤：*

## 運作規則
1. **優先檢索**：當分析任務關鍵字時，應「優先」從上方列表找尋符合的詞彙。
2. **自動擴充**：若現有清單無合適詞彙，可根據內容自創，並建議使用者是否將新詞加入此清單。
3. **一致性**：確保 Things 3 的 Tag 命名與此列表完全相同。
