## � 勞資會議程序（來源：Elizabeth 2025.06.06 筆記）

> 檔案位置：`/Users/leegary/個人app/Inbox/02員工手冊-主管使用（可編輯版）/勞資會議程序事項（2025.06.06.Elizabeth）(1).docx`

### 基本資訊
- **頻率**: 每年 **3月、6月、9月、12月** 開會
- **時間**: 通常上旬，早上一小時
- **屆次**: 目前為第三屆（範例：第三屆第二次 = 2025年6月）

### 會議準備時間線

| 時間點 | 任務 |
|--------|------|
| **會議 2 個月前** | 向淑錡了解委員檔期，找出 2-3 個潛在時間 |
| **敲定時間後** | LINE 全體委員，請選可開會時間（限時回覆，如隔天中午12點前） |
| **會議 7 天前** | LINE 全體委員，請於指定日期前提出議題給**梅芳組長**彙整 |
| **議題收齊後** | 將蒐集到的議題交給**機構主管**表示意見 |
| **會議 3 天前** | 將決定的議題（照片檔）傳給全體委員（群組） |
| **開會前** | 準備紙本：議題（含附件）9份 + 簽到簿 4 份（標題區分四個機構） |

### 開會當天流程

#### 提前 1 小時準備
1. 借錄音筆
2. 排桌椅（9 張椅子）
3. 泡茶（9 杯 + 多一壺）
4. 開空調
5. 每個位子上放一份議題（紙本）
6. 放簽到簿（4 紙，標題區分機構）

#### 開始開會
1. 請出席者簽到
2. 錄音
3. 四個角落拍照 3 張（照到每名出席者）

#### 會議後
1. 製作會議紀錄，請主管審核
2. 會議紀錄（PDF）交給**主席**審核
3. 會議紀錄（4 套，標題區分機構）貼上照片，紙本由主席與記錄蓋章後，連同簽到簿**分存**：
   - **基金會、諮商所** → 人資保存
   - **少家** → 交廖慧雯主任
   - **兒家** → 交廖振杉主任

## 💰 薪資發放規定

在員工手冊中，與發薪日有關的規定主要明訂於以下三份規章與辦法：

### 1. 《工作規則》
- **相關條文**：第十八條（工資計算及發放時間）
- **內容**：「員工工資，經員工同意發放時間如下，如遇例假或休假則提前發放：每月一次：於**每月29日**(二月分於27日)發放當月之工資。」

### 2. 《人事管理辦法》
- **相關條文**：第廿三條
- **內容**：「本會暨附設機構員工薪資發放日為**每月29日**（若逢例假日則提前一或二日為原則，二月份於月底前二日發給）。」

### 3. 《員工敘薪辦法》
- **相關條文**：（針對薪資撥款與明細的進階說明）
- **內容**：
  1. 「月薪於**每月29日**（若逢例假日則提前一或二日為原則；二月份於月底前二日），撥入本會指定銀行個人薪資帳戶。」
  2. 「出納應於每月發薪後，製作個人薪資明細電子檔電傳員工知悉。」

## 📅 家務會議出席規定

根據各機構**《組織章程》**與**《人事管理辦法補充說明》**，規範如下：

### 1. 召開頻率與參與對象（依組織章程）
- **召集人**：各家園主任
- **召開頻率**：每月至少一次
- **一律出席對象**：
  - **兒家**：教保組員工、諮商心理師。
  - **少家**：教保組員工。
- **得派員列席對象**（自由參與）：另一個家園、諮商所，及基金會各組（包含行政組、圖書教育組、研發暨社資組、人資公關組）。

### 2. 排休與出勤規則（依人事管理辦法補充說明）
- **未排休者**：一律參加相關會議。
- **排休者**：
  1. **原則可不參加**（惟對會議決議事項須自行負責追蹤）。
  2. 若排休或非工作時間參加者，依會議時間**發給加班費**。
- *(註：上述排休出勤規則不僅適用於家務會議，亦適用於：會務會議、勞資會議、自辦或學校辦理之親師座談會(IEP)、個案研討等核心會議。)*
