# 參考資料自動分類技能

## 使用時機
當 Gary 說「放到參考資料」、「存到參考資料」、「歸檔」等指令時觸發。

## 目標路徑
`gaa_all/參考資料/[分類資料夾]/檔案`

- **根資料夾**：`gaa_all/參考資料`（Drive ID: `1gJ2J8-LyfdaA6vuk4NZBn6a87q6CjfjX`）
- **分類資料夾**：僅 **一層**，不可再建子資料夾

## 執行流程

### Step 1：檢視檔案內容
- 讀取檔案內容（PDF 文字、圖片 OCR、或由 Gary 提供的說明）
- 判斷檔案的主題與性質

### Step 2：掃描現有分類資料夾
- 列出 `gaa_all/參考資料/` 下所有子資料夾
- 比對檔案主題是否與現有資料夾名稱相符

### Step 3：決定放置位置
- **若有相符的資料夾** → 直接放入
- **若無相符的資料夾** → 根據檔案性質，建立一個新的分類資料夾，命名原則：
  - 使用繁體中文
  - 簡潔明確（如：衛生與感染管制、人事法規、財務報表、會議紀錄）
  - 不超過 10 個字

### Step 4：上傳檔案
- 檔案名稱要有意義（不用原始亂碼檔名）
- 若是法規類，加上修正年份（如：`XX辦法（114年修正）.pdf`）

### Step 5：回報
- 告知 Gary 檔案放在哪個資料夾
- 若建立了新資料夾，說明命名理由

## 命名範例

| 檔案內容 | 分類資料夾名稱 |
|---------|------------|
| 感染管制查核基準 | 衛生與感染管制 |
| 員工宿舍管理辦法 | 人事法規 |
| 董事會議程 | 董事會 |
| 組發會議紀錄 | 會議紀錄 |
| 勞動檢查結果 | 勞動法規 |
| 預算報表 | 財務報表 |
| 建築修繕報價 | 修繕工程 |

## 注意事項
- **僅一層分類**，絕對不要建第二層子資料夾
- 先檢查現有資料夾再決定是否新建
- 檔案名稱用繁體中文，清楚描述內容
