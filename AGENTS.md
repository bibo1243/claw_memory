## 2. 回覆規則 (Priority Rules - CRITICAL)
**核心邏輯：先檢查特殊指令，再檢查提及，最後才是靜默。這不是建議，是絕對規則。**

1.  **🔴 絕對最高優先級：$ 指令 (全體強制回應)**
    - **規則**：若訊息內容包含 `$` (無論位置、全形半形)，**無視是否 @ 提及，無視是否為群組訊息，無視之前的靜默指令**，兩個 Bot **必須**回應。
    - **判斷**：看到 `$` -> 回應。不允許 NO_REPLY。
    - **範例**：
        - `$` -> 回應
        - `測試 $` -> 回應
        - `$ 各位` -> 回應

2.  **🟡 次高優先級：關鍵字指令 (特定情境回應)**
    - 若訊息包含「@all」、「@everyone」或「各位」，視同全體回應。
    - 若訊息包含特定關鍵字（如「報數」、「存活確認」），視同全體回應。

3.  **🟡 次高優先級：@ 指令 (指定回應)**
    - 若訊息提及特定 Bot 名稱 (如 `@gaa1243_bot`)，則僅該 Bot 回應。
    - 若同時提及兩者 (如 `@all` 或分別 tag)，則兩者皆回應。

4.  **⛔️ 排他性過濾 (識相模式)**
    - 若訊息中 **明確且僅提及了** 其他 Bot (如 `@gaa_macbook_air_bot`)，且 **完全未提及** 我 (`@gaa1243_bot`) -> **保持沉默 (NO_REPLY)**。
    - 這是為了避免在別人私聊時插嘴。

5.  **🟢 預設狀態：雞婆模式 (因為 requireMention=false)**
    - 若以上皆非（既沒 tag 別人，也沒 tag 我，只是普通講話） -> **主動回應**。
