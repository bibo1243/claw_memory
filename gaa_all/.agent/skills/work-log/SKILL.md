# 工作日誌技能

## 存放位置

`work-logs/YYYY-MM-DD.md`

## 日誌格式

```markdown
# 工作日誌 — YYYY/MM/DD（星期）

## 今日重點

- 重點一
- 重點二

## 會議/討論

### [會議名稱]
- 討論內容
- 決議事項

## 待辦事項

- [ ] 待辦一
- [ ] 待辦二

## 備註

- 其他補充
```

## 使用方式

### 新增內容
使用者說「記到日誌：XXX」或「今天做了 XXX」
→ 追加到當天的 work-logs/YYYY-MM-DD.md

### 查看日誌
使用者說「今天的工作日誌」「昨天做了什麼」
→ 讀取對應日期的檔案

### 整理對話
使用者轉傳群組對話並說「整理成工作日誌」
→ 摘要重點並追加到當天日誌

### 每日回顧
每天結束時可詢問：「要整理今天的工作日誌嗎？」

## 原則

1. **簡潔為主** — 只記重點，不記流水帳
2. **分類清楚** — 會議、待辦、備註分開
3. **可追溯** — 重要決策要記錄原因
4. **每日一檔** — 方便查找特定日期
